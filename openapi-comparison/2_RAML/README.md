````markdown
# RAML (RESTful API Modeling Language)

Tập trung vào tính kế thừa và mô hình hóa dữ liệu chặt chẽ.

### 🛠 Công cụ hỗ trợ

1. **RAML to HTML**: Chuyển đổi file `.raml` thành tài liệu web.
2. **API Workbench**: Plugin cho các IDE.

### 🚀 Hướng dẫn chạy Demo

### 🚀 Cách chạy

1. Dùng công cụ `raml2html` để xuất bản tài liệu API cho team Frontend.

**1. Chuyển đổi sang tài liệu web:**

```bash
npm install -g raml2html
raml2html library.raml > index.html
```
````
